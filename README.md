# Alvarado Tierras — Informes de Ambientación (mockup)

Landing page mockup para una nueva unidad de negocio de [Alvarado Tierras](https://alvaradotierras.com): generación automática de **informes de ambientación de campos** a partir de un archivo KML.

## Producto

El comprador o vendedor sube el KML del campo, paga entre USD 20 y USD 199 según hectáreas, y recibe un PDF profesional con la ambientación del lote en aproximadamente 30 minutos. Combina datos de elevación (Copernicus), índices de vegetación (Sentinel-2), clima (NASA POWER) y capa de suelos.

## Archivos

- `index.html` — versión v2 (principal): incluye mapa Leaflet interactivo, toggle email/WhatsApp, scroll reveal y tabla comparativa.
- `mockup-v1.html` — versión v1 original, preservada como referencia histórica.

## Stack

- HTML estático (sin build step).
- [Leaflet](https://leafletjs.com/) vía CDN para el mapa interactivo.
- Google Fonts vía CDN para tipografía.
